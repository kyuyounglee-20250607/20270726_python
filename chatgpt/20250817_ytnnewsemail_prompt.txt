1. 뉴스기사에서 
  너는 웹 크롤링 전무가야 
  https://www.ytn.co.kr/_ln/0103_202508170529490007
  뉴스기사 사이트에서 "YTN  이름 (이메일) " 형태의 문장에서만 이름과 이메일을 각각   추출해줘
  문장중에서 이름이 여러개 나올수 있지만 항상 "YTN  이름 (이메일) " 형태에서만
  추출하도록 다시 검토해서 작성해줘
  실행은  VSCODE에서 할수 있도록

2. YTN에서 뉴스링크들을 추출하는 함수
 너는 웹 크롤링 시스템 ytn 뉴스 메인 페이지에서 뉴스 링크만 수집해서 
 뉴스목록은 동적으로 생성되고 있어
 뉴스목록이 로딩될때까지 충분히 여유를 두고
 웹드라이브는 코드로 설치할수 있도록 하고
 리스트에 저장하는 코드 만들어줘(vscode 에서 실행)
 모든 a 태그의  href 들을 수집해줘(/_ln/ 이 포함된 링크만)
 


2번으로 추출한 링크들을 순환문으로
  순환하면 각 url을 1번 함수에 넣어서
  [ (홍길동, aaa@ytn.com), (    ,    ) ]



3. 이 데이터를 엑셀로 저장
이름     |  이메일
홍길동     abc@ytn.com
	

데이터를 엑셀에 저장하기 위해서 다음과 같은 프롬프트를  chatgpt 에 입력하고 코드 확인

 너는 엑셀과 파이썬 전문가야 우리는 [ ( 이름, 이메일주소 ), ... ] 형태의 데이터를 가지고 있고 이 데이터를 읽어서 엑셀에 저장하는 코드를 만들어줘 저장양식은 : 이름 | 이메일 홍길동 abc@abc.dom


이번에는 엑셀에서 데이터를 읽어서 변수에 저장하는 프롬프트

너는 엑셀 파이썬 전문가야 엑셀에 특정 시트에서 데이터를 읽으려고 하는데 양식은 이름 | 이메일 홍길동 abc@abc.com 형태이고 첫라인이 헤더이기때문에 두번째 라인부터 읽어서 [ (홍길동, abc@abc.com), .... ] 형식으로 저장해 주는 코드 만들어줘
