파이썬 반복문 완전 정복

for와 while을 기본부터 고급까지, 단계별로 학습하고 비교해 보세요.

For 문 - 기본(Basic)

1. 리스트 순회: 리스트의 모든 요소를 순서대로 접근합니다.

fruits = ["사과", "바나나", "딸기"]
for fruit in fruits:
    print(fruit)

2. 문자열 순회: 문자열의 각 문자를 하나씩 순회합니다.

my_string = "Python"
for char in my_string:
    print(char)

3. `range(n)`: 0부터 n-1까지의 정수를 생성하여 정해진 횟수만큼 반복합니다.

for i in range(3):
    print(i)

4. `range(start, end)`: start부터 end-1까지의 정수를 생성합니다.

for i in range(2, 5):
    print(i)

5. `range(start, end, step)`: step만큼 증가/감소하며 정수를 생성합니다.

for i in range(10, 0, -2):
    print(i)

6. 튜플 순회: 리스트와 마찬가지로 튜플의 요소를 순회합니다.

my_tuple = (10, 20, 30)
for num in my_tuple:
    print(num)

7. `continue` 문: 특정 조건을 만족할 때 현재 반복을 건너뛰고 다음 반복으로 넘어갑니다.

for i in range(5):
    if i == 2:
        continue
    print(i)

8. `break` 문: 특정 조건을 만족할 때 반복문을 완전히 종료합니다.

for i in range(5):
    if i == 3:
        break
    print(i)

9. `for...else` 구문: `break` 없이 반복문이 완료되면 `else` 블록이 실행됩니다.

for i in range(3):
    print(i)
else:
    print("반복 완료")

10. `for`와 `if`를 이용한 필터링: 조건문을 사용하여 원하는 요소만 처리합니다.

numbers = [1, 2, 3, 4, 5]
for num in numbers:
    if num % 2 == 0:
        print(f"{num}은 짝수입니다.")

For 문 - 중급(Intermediate)

1. 딕셔너리 순회 (키): 딕셔너리 `d`를 `for key in d` 형태로 순회하면 키만 순회합니다.

my_dict = {'a': 1, 'b': 2}
for key in my_dict:
    print(key, my_dict[key])

2. 딕셔너리 순회 (값): `d.values()`를 사용해 값만 순회합니다.

for value in my_dict.values():
    print(value)

3. 딕셔너리 순회 (키, 값): `d.items()`를 사용해 키와 값을 동시에 순회합니다.

for key, value in my_dict.items():
    print(f"키: {key}, 값: {value}")

4. `enumerate()`: 인덱스와 요소를 함께 순회합니다.

colors = ["red", "green", "blue"]
for index, color in enumerate(colors):
    print(f"인덱스 {index}: {color}")

5. `enumerate(start=1)`: 인덱스를 1부터 시작하도록 설정합니다.

for index, color in enumerate(colors, start=1):
    print(f"{index}번째: {color}")

6. `zip()`: 여러 이터러블 객체를 병렬로 순회합니다.

names = ["A", "B"]
ages = [20, 30]
for name, age in zip(names, ages):
    print(f"{name}는 {age}살입니다.")

7. 중첩 반복문: 반복문 안에 또 다른 반복문이 있는 구조로, 2차원 데이터를 다룰 때 사용됩니다.

for i in range(3):
    for j in range(2):
        print(f"({i}, {j})")

8. 리스트 컴프리헨션: 반복문을 한 줄로 압축하여 새로운 리스트를 만듭니다.

numbers = [1, 2, 3]
squared_numbers = [num * num for num in numbers]
print(squared_numbers)

9. 리스트 컴프리헨션 + `if`: 조건문을 추가하여 원하는 요소만 필터링합니다.

numbers = [1, 2, 3, 4, 5]
even_numbers = [num for num in numbers if num % 2 == 0]
print(even_numbers)

10. `for`와 `*` 연산자: 리스트 언팩킹을 활용한 순회입니다.

pairs = [(1, 'a'), (2, 'b')]
for num, char in pairs:
    print(num, char)

For 문 - 고급(Advanced)

1. 중첩 리스트 컴프리헨션: 2차원 리스트를 평탄화(flatten)하는 예제입니다.

matrix = [[1, 2], [3, 4]]
flattened = [num for row in matrix for num in row]
print(flattened)

2. 딕셔너리 컴프리헨션: `for` 루프를 사용해 딕셔너리를 한 줄로 생성합니다.

squared_dict = {x: x*x for x in range(3)}
print(squared_dict)

3. 파일 순회: 파일을 한 줄씩 읽어와 처리할 때 사용합니다. (실제 파일이 필요)

# with open('data.txt', 'r') as file:
#     for line in file:
#         print(line.strip())

4. 제너레이터 표현식: `()`를 사용해 메모리를 절약하는 제너레이터를 만듭니다.

numbers = (x*x for x in range(5))
for num in numbers:
    print(num, end=' ')

5. 역순 순회: `reversed()` 함수를 사용해 역순으로 순회합니다.

my_list = [1, 2, 3]
for num in reversed(my_list):
    print(num)

6. `for` 루프 내 `try...except`: 반복 중 발생하는 예외를 개별적으로 처리합니다.

items = [1, 2, 'a', 4]
for item in items:
    try:
        print(item * 2)
    except TypeError:
        print(f"'{item}'은 숫자가 아닙니다.")

7. 인덱스를 이용한 값 변경: `range()`와 인덱싱을 이용해 리스트의 값을 직접 변경합니다.

my_list = [1, 2, 3]
for i in range(len(my_list)):
    my_list[i] *= 2
print(my_list)

8. 여러 이터러블 동시 순회 (복수 `zip`): 여러 리스트를 동시에 순회합니다.

l1 = [1, 2]
l2 = [3, 4]
l3 = [5, 6]
for a, b, c in zip(l1, l2, l3):
    print(a, b, c)

9. `for`와 제너레이터 체인: 여러 제너레이터를 연결하여 순회합니다.

from itertools import chain
list1 = ['a', 'b']
list2 = ['c', 'd']
for item in chain(list1, list2):
    print(item)

10. `itertools`의 `product`로 다중 루프 대체: 중첩 루프를 한 줄로 표현합니다.

from itertools import product
for i, j in product(range(2), range(2)):
    print(f"({i}, {j})")

While 문 - 기본(Basic)

1. 기본 카운터 루프: 조건이 참인 동안 코드를 반복 실행합니다.

count = 0
while count < 3:
    print(f"카운트: {count}")
    count += 1

2. 무한 루프와 `break`: `while True`로 무한 루프를 만들고 특정 조건에서 `break`로 종료합니다.

while True:
    user_input = input("종료하려면 'exit'을 입력하세요: ")
    if user_input == 'exit':
        print("프로그램 종료")
        break

3. 사용자 입력 유효성 검사: 올바른 입력이 들어올 때까지 반복합니다.

password = ""
while len(password) < 8:
    password = input("8자 이상 비밀번호를 입력하세요: ")
print("비밀번호 설정 완료.")

4. 리스트의 요소 삭제: 조건에 맞는 요소를 모두 삭제할 때 유용합니다.

numbers = [1, 2, 3, 2, 4]
while 2 in numbers:
    numbers.remove(2)
print(numbers)

5. `while...else` 구문: `while` 문이 `break` 없이 종료되면 `else` 블록이 실행됩니다.

count = 0
while count < 3:
    print(count)
    count += 1
else:
    print("조건이 거짓이 되어 루프 종료")

6. `continue` 문: 특정 조건을 만족할 때 현재 반복을 건너뛰고 다음 반복으로 넘어갑니다.

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)

7. 플래그(flag) 변수 사용: 루프의 상태를 제어하는 변수를 사용합니다.

running = True
while running:
    user_input = input("정지하려면 'stop' 입력: ")
    if user_input == 'stop':
        running = False

8. 게임 루프: 간단한 게임에서 종료 조건이 만족될 때까지 반복합니다.

import random
secret_number = random.randint(1, 5)
guess = 0
while guess != secret_number:
    guess = int(input("1에서 5 사이의 숫자를 맞춰보세요: "))
print("정답입니다!")

9. 문자열 인덱싱: 인덱스를 사용해 문자열의 특정 부분을 순회합니다.

text = "hello"
i = 0
while i < len(text):
    print(text[i])
    i += 1

10. 리스트의 마지막 요소부터 순회: `pop()` 메서드를 사용해 리스트를 비우면서 순회합니다.

my_list = [10, 20, 30]
while my_list:
    print(my_list.pop())

While 문 - 중급(Intermediate)

1. 다중 조건 `while` 문: 여러 조건을 `and`, `or`로 묶어 루프를 제어합니다.

x = 0
y = 10
while x < 5 and y > 5:
    print(f"x={x}, y={y}")
    x += 1
    y -= 1

2. 스택(Stack) 구현: `while` 문으로 스택의 요소를 모두 비울 때 사용합니다.

stack = [1, 2, 3]
while stack:
    print(stack.pop())

3. 리스트에서 조건에 맞는 요소만 추출: 리스트를 순회하며 특정 조건을 만족하는 요소만 새로운 리스트에 담습니다.

nums = [1, 2, 3, 4, 5]
i = 0
evens = []
while i < len(nums):
    if nums[i] % 2 == 0:
        evens.append(nums[i])
    i += 1
print(evens)

4. 큐(Queue) 구현: `collections.deque`와 함께 `while` 문으로 큐를 처리합니다.

from collections import deque
queue = deque(['A', 'B', 'C'])
while queue:
    print(queue.popleft())

5. 숫자 자리수 합계 구하기: 숫자를 10으로 나누는 연산을 반복하여 각 자리수를 얻습니다.

num = 1234
sum_of_digits = 0
while num > 0:
    sum_of_digits += num % 10
    num //= 10
print(sum_of_digits)

6. 문자열에서 특정 부분 문자열 찾기: `find()` 메서드와 함께 `while` 문을 사용하여 모든 발생 위치를 찾습니다.

text = "banana banana"
index = -1
while True:
    index = text.find('na', index + 1)
    if index == -1:
        break
    print(f"found at index: {index}")

7. 재시도 로직 구현: 네트워크 요청 등 실패할 수 있는 작업을 재시도할 때 사용합니다.

import time
attempts = 0
max_attempts = 3
success = False
while attempts < max_attempts and not success:
    print(f"시도 횟수: {attempts + 1}")
    # 가상의 작업 (실패)
    time.sleep(1)
    attempts += 1
    if attempts == 2: # 2번째 시도에서 성공했다고 가정
        success = True
if success:
    print("작업 성공!")
else:
    print("작업 실패...")

8. 중첩 `while` 문: 반복문 안에 또 다른 `while` 문을 사용합니다.

i = 0
while i < 2:
    j = 0
    while j < 2:
        print(f"({i}, {j})")
        j += 1
    i += 1

9. `while` 루프와 `continue`를 이용한 필터링: `for` 루프와 비슷하게 특정 조건을 건너뛸 수 있습니다.

i = 0
numbers = [1, 2, 3, 4, 5]
while i < len(numbers):
    if numbers[i] % 2 == 0:
        i += 1
        continue
    print(f"홀수: {numbers[i]}")
    i += 1

10. `while`을 이용한 리스트 병합: 두 개의 정렬된 리스트를 `while` 루프를 사용해 병합합니다.

list1 = [1, 3, 5]
list2 = [2, 4, 6]
merged_list = []
i = 0
j = 0
while i < len(list1) or j < len(list2):
    if i < len(list1) and (j >= len(list2) or list1[i] <= list2[j]):
        merged_list.append(list1[i])
        i += 1
    else:
        merged_list.append(list2[j])
        j += 1
print(merged_list)

While 문 - 고급(Advanced)

1. 이진 탐색(Binary Search) 알고리즘: 정렬된 리스트에서 특정 값을 효율적으로 찾습니다.

my_list = [1, 3, 5, 7, 9]
target = 7
low = 0
high = len(my_list) - 1
while low <= high:
    mid = (low + high) // 2
    if my_list[mid] == target:
        print(f"타겟 발견! 인덱스: {mid}")
        break
    elif my_list[mid] < target:
        low = mid + 1
    else:
        high = mid - 1

2. 피보나치 수열 생성: `while` 문을 사용해 지정된 횟수만큼 피보나치 수열을 생성합니다.

a, b = 0, 1
count = 0
limit = 10
while count < limit:
    print(a, end=' ')
    a, b = b, a + b
    count += 1

3. 유클리드 호제법(최대공약수): `while` 문을 사용해 두 수의 최대공약수를 구합니다.

5. 숫자 야구 게임: 사용자가 정답을 맞출 때까지 반복되는 복잡한 게임 로직에 사용합니다.

import random
answer = random.sample(range(1, 10), 3)
strikes = 0
while strikes < 3:
    guess_str = input("3자리 숫자를 입력하세요: ")
    guess = [int(i) for i in guess_str]
    strikes = sum(1 for i, g in enumerate(guess) if answer[i] == g)
    balls = sum(1 for g in guess if g in answer) - strikes
    print(f"스트라이크: {strikes}, 볼: {balls}")

i = 0

    i += 1

7. 동적 조건 루프: 루프 조건이 루프 내에서 동적으로 변경되는 경우입니다.

data = [1, 2, 3]
i = 0

        data.append(4) # 리스트 크기 변경

8. 재귀 함수 대체: 재귀 호출을 `while` 루프로 바꾸어 스택 오버플로우를 방지합니다.

# 재귀 팩토리얼
# def factorial_recursive(n):
#     if n == 0: return 1
#     return n * factorial_recursive(n-1)

# while 루프 팩토리얼
n = 5
result = 1
while n > 0:
    result *= n
    n -= 1
print(result)

9. `while` 문을 이용한 버블 정렬: 정렬이 완료될 때까지 `while` 루프를 반복합니다.

my_list = [5, 1, 4, 2, 8]
n = len(my_list)
swapped = True
while swapped:
    swapped = False
    for i in range(n - 1):
        if my_list[i] > my_list[i+1]:
            my_list[i], my_list[i+1] = my_list[i+1], my_list[i]
            swapped = True
print(my_list)

10. `while` 루프와 제너레이터 `yield`: 제너레이터 함수 내에서 `while`을 사용해 값을 생성합니다.

def my_generator(limit):
    count = 0
    while count < limit:
        yield count * 2
        count += 1

gen = my_generator(3)
for num in gen:
    print(num)

For vs While 비교

`for` 문과 `while` 문은 모두 반복을 수행하지만, 사용되는 상황이 다릅니다. 둘 중 어떤 것을 사용할지 결정하는 것이 중요합니다.

For 문

  • 주요 용도: 반복 횟수가 명확하거나, 리스트, 튜플, 딕셔너리, 문자열과 같은 **이터러블 객체의 모든 요소**를 순회할 때 사용합니다.
  • 구조: `for` 변수 `in` 이터러블 객체:
  • 코드 예시:
    • # 리스트의 모든 요소 순회
numbers = [1, 2, 3]
for num in numbers:
    print(num)

# 3번 반복
for _ in range(3):
    print("hello")

While 문

  • 주요 용도: 반복 횟수가 불확실하고, 특정 **조건이 참인 동안** 반복을 계속할 때 사용합니다.
  • 구조: `while` 조건:
  • 코드 예시:
    • # 조건이 만족될 때까지 반복
count = 0
while count < 3:
    print(count)
    count += 1

# 올바른 입력이 나올 때까지 반복
password = ""
while len(password) < 8:
    password = input("8자 이상 비밀번호: ")

최종 퀴즈
